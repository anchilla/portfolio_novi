Description:
This web application is developed for PC devices. It represents a calculator, meant to be used for mathematical operations. You can delete numbers or clear all. It's very easy to use.
It was developed entirely with React.js and contains custom components logic.

Problems and solutions:
The problem that I had with this project was making the calculator show the result of operation.
I tried a lot of things, but what ultimately worked was making a global variable to corespond to different operations and actions the user is taking. And to use those different cases to perform various commands.
